# AstronvimUser
My user config for Astronvim. Clone into lua/user directory.

### OS Dependencies
- languages
    - Rust
    - Go
    - NodeJS and NPM
    - TS-Server
    - Git
- image.nvim
    - Lua 5.1
    - luarocks
    - magick LuaRock
    - ImageMagick
    - Kitty or Wezterm
    - curl
- obsidian.nvim
    - Obsidian (recommended)
- nvim-spectre
    - ripgrep
    - devicons
    - sed
- And all other AstroNvim recommended plugins...

```sh
git clone https://github.com/john-okeefe/AstronvimUser.git ~/.config/nvim/lua/user/
```
